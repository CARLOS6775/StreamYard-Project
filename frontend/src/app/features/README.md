# Features

Contiene los módulos de negocio de la aplicación Angular:

- **auth/**: Login, registro, recuperación de contraseña
- **studio/**: Estudio de grabación (interfaz principal)
- **broadcasts/**: Gestión de transmisiones en vivo
- **chat/**: Componentes de chat durante transmisiones
- **profile/**: Perfil de usuario y configuraciones
- **dashboard/**: Panel de control con estadísticas
- **media/**: Galería multimedia y gestión de contenido

Cada feature es un módulo independiente con sus propios componentes, servicios y rutas.
