# Core

Contiene el código fundamental y reutilizable de la aplicación Angular:

- **Models**: Modelos de datos e interfaces TypeScript
- **Services**: Servicios core (HTTP, auth, storage)
- **Interceptors**: Interceptores HTTP para JWT, logging, errores
- **Guards**: Guards de rutas para autenticación y permisos
- **Pipes**: Pipes globales de transformación y validación
- **Directives**: Directivas compartidas

Este código es utilizado por toda la aplicación y no depende de módulos de negocio específicos.
