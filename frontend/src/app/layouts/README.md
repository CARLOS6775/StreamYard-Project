# Layouts

Contiene los layouts principales de la aplicación Angular:

- **auth**: Layout para páginas de autenticación (login, registro)
- **main**: Layout principal para la aplicación autenticada
- **studio**: Layout especial para el estudio de grabación
- **public**: Layout para páginas públicas

Cada layout define la estructura base de las páginas que incluye:
- Header/Navegación
- Sidebar (si aplica)
- Footer
- Área de contenido principal
