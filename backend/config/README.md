# Config

Contiene toda la configuración del backend:

- **database.config.ts**: Configuración de conexión a base de datos
- **jwt.config.ts**: Configuración de autenticación JWT
- **redis.config.ts**: Configuración de Redis para caching y sesiones
- **mediasoup.config.ts**: Configuración de WebRTC (Mediasoup)
- **ffmpeg.config.ts**: Configuración de FFmpeg para procesamiento de video
- **app.config.ts**: Configuración general de la aplicación
