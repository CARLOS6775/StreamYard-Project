# Common

Contiene código compartido utilizado en toda la aplicación backend:

- **Decoradores**: Decoradores personalizados de NestJS
- **Guards**: Guards de autenticación y autorización
- **Interceptors**: Interceptores para logging, transformación de respuestas
- **Pipes**: Pipes de validación y transformación de datos
- **Interfaces**: Interfaces y tipos compartidos
- **Utils**: Funciones utilitarias reutilizables
