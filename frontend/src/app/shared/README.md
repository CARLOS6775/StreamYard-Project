# Shared

Contiene componentes y elementos de UI reutilizables:

- **Components**: Botones, inputs, modales, cards, etc.
- **Pipes**: Pipes de formato (fecha, moneda, truncado)
- **Directives**: Directivas de UI (auto-focus, click-outside)
- **Modules**: Módulos de componentes compartidos

Los componentes en este directorio son genéricos y pueden ser utilizados en cualquier parte de la aplicación sin dependencias específicas del negocio.
