# Modules

Contiene los módulos de negocio de la aplicación:

- **auth/**: Módulo de autenticación y registro de usuarios
- **broadcasts/**: Gestión de transmisiones en vivo
- **chat/**: Sistema de chat durante las transmisiones
- **users/**: Gestión de perfiles de usuario
- **media/**: Procesamiento y almacenamiento multimedia
- **notifications/**: Sistema de notificaciones
- **analytics/**: Estadísticas y análisis de transmisiones

Cada módulo contiene su propia lógica de negocio, controladores, servicios y DTOs.
